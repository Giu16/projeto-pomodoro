# projeto-pomodoro

PomoRitual
Um timer Pomodoro intuitivo e acessível para manter seu ritmo produtivo.

Sobre o projeto:

PomoRitual é uma ferramenta que vai além da contagem regressiva simples, combinando simplicidade funcional, interatividade refinada e uma interface fluida para sustentar seu foco e produtividade no dia a dia.

Funcionalidades principais:

Timer Pomodoro com notificações discretas e adaptativas.

Interface responsiva e acessível, com suporte total à navegação por teclado e atributos ARIA.

Animações SVG elegantes para feedback visual dinâmico.

Design leve e fluido, pensado para não interromper seu fluxo mental.

Tecnologias usadas:

JavaScript (ES6+)

HTML5 e CSS3 (Flexbox, variáveis CSS, animações)

Web Notifications API

SVG animado com CSS
